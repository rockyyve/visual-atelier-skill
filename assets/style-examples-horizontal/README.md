# Static 4:3 Landscape Style Examples

These PNG files are the built-in high-fidelity landscape style examples for this skill.

Use them as the style selection surface when the user asks for 4:3 landscape images. Do not regenerate per-request demo images unless the user explicitly asks for fresh visual exploration.

## Styles

1. `style-01-clean-lifestyle-guide-4x3.png` - bright, practical, friendly guide layout rebuilt for a wide canvas.
2. `style-02-signal-playbook-4x3.png` - sharp action-playbook layout with landscape diagnostic panels and command ribbons.
3. `style-03-editorial-magazine-4x3.png` - refined magazine spread with editorial columns and source-object composition.
4. `style-04-tech-dashboard-4x3.png` - structured wide dashboard for technical workflows and system knowledge.
5. `style-05-handdrawn-notebook-4x3.png` - warm landscape notebook spread with taped diagrams and sticky checkpoints.
6. `style-06-ai-tool-lab-4x3.png` - clean AI tool lab landscape card with horizontal experiment lanes.
7. `style-07-daily-lime-lab-4x3.png` - white-green Skill Daily landscape briefing with title, reason cards, and product object.
8. `style-08-obsidian-neon-knowledge-4x3.png` - dark neon knowledge-vault landscape card with graph, note cards, and insight strip.

All examples in this directory are `1280 x 960` PNGs.
