# Static Style Examples

These PNG files are the built-in high-fidelity style examples for this skill.

Use them as the style selection surface before final image generation. Do not regenerate per-request demo images unless the user explicitly asks for fresh visual exploration.

## Styles

1. `style-01-clean-lifestyle-guide.png` - bright, practical, friendly guide cards for productivity, learning, light tutorials, and save-worthy lists.
2. `style-02-signal-playbook.png` - sharp action-playbook cards for technical workflows, mistakes, deployment, checks, and decision guides.
3. `style-03-editorial-magazine.png` - refined magazine cards for product analysis, deep thinking, course notes, and premium explainers.
4. `style-04-tech-dashboard.png` - structured technical dashboard cards for AI tools, coding tutorials, systems, and data-heavy knowledge.
5. `style-05-handdrawn-notebook.png` - warm notebook cards for beginner explainers, mental models, learning notes, and practical checklists.
6. `style-06-ai-tool-lab.png` - clean AI tool lab cards for Codex, agents, MCP, skills, automation, and AI workflow notes.
7. `style-07-daily-lime-lab.png` - white-green Skill Daily cards for AI skill recommendations, tool daily posts, frontend/design resources, open-source project intros, plugins, and templates.

All examples are `1080 x 1350` PNGs.
